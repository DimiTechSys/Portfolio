# Robotique & Systèmes embarqués — Dimitri Voissier

Projets en **systèmes embarqués, robotique et IoT**, réalisés dans le cadre de ma formation
d'ingénieur à l'ESIEA (**majeure Systèmes Embarqués & Autonomes**). Ce dossier contient aussi
mon portfolio académique en page unique (React/Vite).

## 🤖 Projets

### Casier intelligent connecté — ESP32 + Application Web
Casier sécurisé piloté par **ESP32**, avec développement complet de l'interface web (front-end
et back-end), communications sans fil et **API embarquée**. Pont entre l'électronique embarquée
et le développement logiciel.

### Camion de pompier robotisé — Raspberry Pi
Prototype robotisé : contrôle des moteurs, intégration de capteurs et **commande à distance**.
Travail d'équipe et débogage en temps réel.

### Maison intelligente — Arduino / IoT
Système domotique complet : automatisation de l'éclairage, surveillance environnementale,
fonctions de sécurité, collecte de données capteurs et **communication IoT**.

## 🛠️ Compétences mises en œuvre

- **Cartes & robotique** : Arduino, Raspberry Pi, STM32, ESP32, FPGA (ModelSim, Quartus)
- **Langages** : C, C++, Python, VHDL, VAL3
- **Web embarqué** : interface front/back, API, communication sans fil
